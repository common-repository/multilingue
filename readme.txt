Translate your website in one click !

== Description ==

Allow your visitors to translate the pages with one click!

Translations available : English, French, German, Italian, Spanish, Chinese.

Most in 1 click: no configuration, automatic detection of the original language, translation by Google Translate.

== Installation ==

1. Install plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget in the 'Widgets' menu
4. That's it
